사용 기술

  • RAG를 활용한 문서 분석 및 내용 보강

LangChain 기반 RAG 활용 챗봇

프로젝트 기간

  • 2024년 7월 ~ 2024년 9월

목표

사용 기술

  • Python, LangChain, FastAPI, FAISS, OpenAI

핵심 기능

  • 자연어 처리 기반 대화

성과

  • 데이터 정제 및 검색 최적화 → RAG 성능 개선
  • 벡터 DB 활용으로 검색 속도 향상
