LangChain 기반 RAG 활용 챗봇

LangChain과 RAG 시스템을 활용한 AI 기반의 자연어 처리 챗봇 프로젝트입니다. 이 챗봇은 사용자 질문에 대한 실시간 답변을 제공하며, 텍스트 벡터화를 사용해 데이터 검색 기능을 포함하고 있습니다.

프로젝트 기간: 2024년 7월 ~ 진행 중

프로젝트 목표: LangChain과 RAG 시스템을 사용해 사용자에게 카카오 부트캠프에 대한 정보를 실시간 답변을 제공하는 챗봇을 구현.

사용 기술: Python, LangChain, FastAPI, FAISS, OpenAI.

핵심 기능: 실시간 질문 응답, 텍스트 벡터화를 통한 데이터 검색, 자연어 처리 기반 챗봇.

역할: 데이터 수집 및 정제, LLM 활용 로직 설계, 텍스트 벡터화 시스템 설계, 데이터 검색 알고리즘 구현.

성과: 챗봇을 통해 사용자 질의와 답변을 분석하여 RAG의 성능을 높이기 위한 데이터 정제 및 검색 기법 증강을 달성함. 벡터 DB 관리 기술을 활용해 검색 속도를 최적화함.

향후 계획: 추가적인 기능 확장, 사용자 경험 개선 등을 통해 2기 부트캠프 사용자들 대상 상용화 목표.

Gitfolio 프로젝트

GitHub 레포지토리 내용을 기반으로 이력서를 자동으로 생성해주는 프로젝트입니다. 이 프로젝트는 개발자들의 GitHub 활동 내역을 분석하고, 이력서에 필요한 정보를 추출하여 템플릿에 맞춰 이력서를 자동으로 작성하는 시스템을 목표로 했습니다.

프로젝트 기간: 2024년 10월 ~ 진행 중

프로젝트 목표: GitHub 레포 내용을 분석하여 개발자들의 프로젝트 경험을 자동으로 이력서에 반영하는 시스템 구현.

사용 기술: Python, LangChain, MongoDB, GitHub API.

핵심 기능: GitHub API를 활용한 데이터 수집, 프로젝트 정보를 기반으로 자동 이력서 생성, PDF 및 Markdown 이력서 출력.

역할: 데이터 수집 및 정제, 텍스트 분석 알고리즘 설계, 이력서 생성 로직, RAG를 통한 문서 읽기, 랭체인을 통한 유연한 LLM모델 사용 및 용도에 따른 프롬프트 로직 구현.

성과: GitHub 활동 내역을 기반으로 자동으로 이력서를 생성함으로써 개발자들이 이력서를 보다 쉽게 관리할 수 있는 시스템을 구축. PDF 및 Markdown 형식으로 이력서를 자동 생성하는 기능을 구현함.

향후 계획: 추가적인 템플릿 확장, 다양한 포맷의 이력서 제공, 사용자 맞춤형 이력서 추천 기능 개발 계획.

About me

저는 경북대학교에서 아동학을 전공하며 아이들의 성장을 돕고, 교육 프로그램을 기획하는 일에 큰 보람을 느꼈습니다. 졸업 후 3년 동안 보육교사로 일하면서, 아이들의 교육과 발달을 지원하는 일을 하며 다양한 경험을 쌓았습니다. 하지만 빠르게 변화하는 기술의 시대에 아이들을 위한 교육 도구가 충분하지 않다고 느끼게 되었습니다. 특히 인공지능과 같은 기술이 아이들의 성장과 교육에 더 효과적으로 적용될 수 있을 것이라는 생각이 들었고, 이를 직접 해결하고자 개발자로 전향하게 되었습니다.

개발자로서의 첫걸음을 떼며 인공지능과 데이터 분석에 대한 학습을 시작했고, 네이버커넥트재단과 카카오테크 부트캠프를 통해 인공지능 모델링, 데이터 사이언스, 딥러닝 등의 기술을 배웠습니다. 이러한 지식을 바탕으로 MiniBaseball, LangChain 기반 RAG 챗봇, Gitfolio 프로젝트와 같은 실습 프로젝트를 통해 실질적인 개발 역량을 키우고 있습니다.

특히, 아이들과의 경험을 통해 남들과는 다른 시각에서 문제를 바라보고 창의적인 해결책을 모색할 수 있습니다. 아이들의 시선에서 기술을 적용하는 방식은 더욱 풍부한 아이디어와 넓은 시야를 가능하게 하며, 이는 개발 과정에서 새로운 가치를 창출하는 데 중요한 역할을 합니다.

또한, 보육교사로서의 경험을 바탕으로 후진 양성에도 기여할 수 있는 역량을 갖추고 있습니다. 이는 기술을 잘 모르는 고객이나 사용자들에게도 맞춤형으로 설명하고, 그들이 쉽게 이해할 수 있도록 교육하는 데 큰 도움이 됩니다. 앞으로도 아이들을 위한 혁신적인 기술과 교육 도구를 개발하고, 더 나아가 세상을 변화시키는 기술적 해결책을 제시하고자 끊임없이 도전할 것입니다.
