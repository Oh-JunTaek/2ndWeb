# 2ndWeb
이력서나 지원서에 첨부할, 혹은 개인의 개발자 발자취를 남기는 목적의 웹 페이지 입니다.
깃허브에서는 이것저것 미완의 프로젝트 및 공부과정등이 모두 나와있지만 본 웹페이지에서는 객관적인 정보나 결과물등이 수록되어 있습니다.
